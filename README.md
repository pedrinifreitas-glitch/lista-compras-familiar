# 🛒 Lista de Compras Familiar

Uma aplicação web para gerir listas de compras familiares com sincronização em tempo real.

## ✨ Funcionalidades

- ✅ Adicionar, editar e eliminar produtos
- 📂 Organização por categorias
- 🔄 Sincronização em tempo real (Firebase)
- 👥 Multi-utilizador
- 📊 Estatísticas em tempo real
- 📱 Design responsivo (mobile-friendly)
- 💾 Exportar lista para TXT

## 🚀 Como Usar

1. Abre a aplicação no teu navegador
2. Introduz o teu nome
3. Começa a adicionar itens à lista
4. Partilha o link com a família!

## 🔧 Configuração do Firebase

Para usar a sincronização em tempo real:

1. Cria um projeto em [Firebase Console](https://console.firebase.google.com/)
2. Ativa o Firestore Database
3. Copia as configurações do projeto
4. Substitui no ficheiro `index.html` na secção `firebaseConfig`

## 📝 Tecnologias

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6+)
- Firebase Firestore

## 👨‍👩‍👧‍👦 Para a Família

Todos os membros podem:
- Ver a lista em tempo real
- Adicionar novos produtos
- Marcar itens como comprados
- Editar quantidades
- Ver quem adicionou cada item
